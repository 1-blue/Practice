# 1. vue-cli설치
`npm install -g @vue/cli`
`npm update -g @vue/cli`
`npm vue --version`
`npm install -g @vue/cli-service-global`   // vue serve실행하려면 설치해야함    
+ global설치 위치 : `%USERPROFILE%\AppData\Roaming\npm\node_modules`
